Blockchain-Based Decentralized NFT Auction Platform

This project is a full stack application that leverages blockchain technology for decentralized auction functionality. The website enables users to mint NFTs, initiate auctions for NFTs, place bids on open auctions, and approve transfers of NFTs. The user interface is integrated with MetaMask and enables users to perform Ethereum transactions directly on their web browser. The back-end consists of smart contract logic deployed to a private Ethereum blockchain that handles the core logic and data storage for the application.

Automated Music Genre Classification using Machine Learning

This machine learning project involves training a neural network using the GTZAN dataset to automate music genre classification tasks. The model achieves an accuracy of 75%, surpassing the human benchmark accuracy rate of 70%. A feedforward neural network is employed on the dataset and is set up in a cloud-based environment (Google Colab) to enhance the model's efficiency. By automating music genre classification, the user experience on music streaming services is significantly improved, as these models enable more accurate music discovery and recommendation processes. This benefits both artists and listeners by ensuring that the right music reaches the right audience.
